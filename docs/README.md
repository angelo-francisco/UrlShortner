# URL SHORTNER
### About
A simple app that when get a url convert it to a short version

### Tools 
- Python
- Libs
    - Customtkinter
    - Requests

### How to run?
```bash
# clone the repository
git  clone https://github.com/angelo-francisco/UrlShortner.git

# create the venv
python -m venv venv

# acess the venv
./venv/Scripts

# install the dependencies
pip install customtkinter requests

# run the project
python src/main.py
```
